==== libMary Roadmap ====

Ultimately, this is meant to be an ideal development environment for server
apps, various backends, soft real-time apps.

Legend:
    v - fully implemented;
    * - partially implemented;
    _ - not started
    M - used from MyNC

[*] Reference counting
    [v] BasicReferenced
    [v] VirtReferenced
    [v] Referenced
    [v] Object
    [v] Ref

    [ ] Comprehensive multithreaded tests
[*] Language support

    [v] Callback

    [M] Pointer
    [v] Memory
